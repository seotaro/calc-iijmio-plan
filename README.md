# calc-iijmio-plan

[IIJmio](https://www.iijmio.jp/) で複数SIMをデータシェアするときに必要な容量とプランの組み合わせを最適化する。

実行

```bash
node main.js {SIM契約数} {音声SIM契約数} {データ容量下限} {データ容量上限}
```
